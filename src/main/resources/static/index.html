Bem-vindo à API Micronectar

Esta é uma API RESTful desenvolvida em Spring Boot para a plataforma Micronectar.

O serviço está em execução, mas esta URL raiz não oferece funcionalidades diretas via navegador. A interação com a API deve ser feita através de clientes HTTP (como Insomnia ou Postman) nos endpoints apropriados.

Endpoints Públicos

Estes endpoints podem ser acessados sem autenticação.

POST /api/auth/register/cliente

Registra um novo usuário do tipo Cliente.

POST /api/auth/register/investidor

Registra um novo usuário do tipo Investidor.

POST /api/auth/register/microempreendedor

Registra um novo Microempreendedor.

POST /api/auth/login

Autentica um usuário e retorna um token JWT.

Endpoints Protegidos

Atenção: Para acessar os endpoints abaixo, é necessário incluir o token JWT obtido no login no cabeçalho da requisição:
Authorization: Bearer <seu_token_aqui>
POST /api/microempreendedores/{id}/produtos

Adiciona um novo produto ao perfil de um microempreendedor. Requer autenticação como o próprio Microempreendedor ou como Admin.

GET /api/microempreendedores/{id}/produtos

Lista todos os produtos de um microempreendedor específico. Requer qualquer tipo de autenticação.

POST /api/microempreendedores/{id}/votos

Registra um voto (promissor ou não) no perfil de um microempreendedor. Requer autenticação como Cliente, Investidor ou Admin.
